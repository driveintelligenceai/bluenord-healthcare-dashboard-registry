# BlueNord Healthcare Dashboard Registry

Public shadcn registry for sanitized healthcare dashboard demos and theme tokens.

## Items

- `healthcare-dashboard-demo`: a compact hospital marketing intelligence dashboard block.
- `knowledge-graph-demo`: a sanitized hospital competitor knowledge graph block.
- `healthcare-theme-tokens`: sanitized BlueNord-style CSS tokens for public demos.

## Checks

```bash
npm install
npm run registry:validate
npm run registry:build
```

This repository ships sample fixture data only. It does not include raw crawls, secrets, private caches, model weights, or patient records.
